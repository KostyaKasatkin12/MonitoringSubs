    📊 Результаты импорта

    📊 Как это работает:

    1. Вы создаете пароль приложения в своем почтовом сервисе
    2. Мы подключаемся к вашей почте через защищенный IMAP
    3. Анализируем письма с помощью ИИ для поиска подписок
    4. Извлекаем информацию о названии, цене и дате платежа
    5. Добавляем найденные подписки в ваш список

    ⏱ Поиск может занять несколько минут в зависимости от объема почты.
